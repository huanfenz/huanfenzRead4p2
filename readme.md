## 项目介绍

基于Android的本地电子书阅读器，实现了书架管理、电子书的导入与阅读、阅读界面个性化设置等功能。

## 环境介绍

Android版本：
推荐：9.0(29)
最低：4.4(19)

## 运行步骤

1、使用Android Studio打开项目，等待下载对应版本的SDK。

2、连接手机，运行项目。
